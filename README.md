# 行動寬頻用戶 SQL 查詢與資料庫管理工具  


- 本專案延續前項Python視覺化與報表自動化流程
- 將所有清洗後的行動寬頻資料彙整進 SQLite 建立本地資料庫  
- 透過 Python 製作查詢介面，用於快速比對、篩選與分析不同年份與業者的數據  

藉由將資料轉換成可查詢的資料庫格式，  
本專案將原本分散在多個 Excel 報表的資料集中化管理，  
形成「資料整理 → 儲存 → 查詢」的完整流程，  
並提供後續視覺化、分析與應用程式串接更彈性的資料來源。

## 技術亮點
- 使用 pandas 完成前置資料清洗與 schema 標準化，使多份來源資料可整併入同一資料表。
- 規劃 SQLite 資料表結構與資料型別，建立統一且具可擴充性的儲存格式。
- 利用 Python sqlite3 執行 SQL 查詢（GROUP BY、聚合分析、條件查詢）。
- 開發 CLI 查詢工具，支援輸入年份／業者並即時取得查詢結果。
